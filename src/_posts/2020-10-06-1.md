submoduleが必要なので注意。
忘れるとconfigureでgoogletestがどうとか警告が出て、
そのままmakeしてもmake checkで失敗する。
あと、そこそこ時間がかかる。
```
git clone https://github.com/google/protobuf.git
cd protobuf
git submodule init
git submodule update
./autogen.sh
./configure
make
make check
# TOTAL: 7
# PASS:  7
make install
ldconfig
protoc --version
```
