sarメモ

2019年01月25日 20:00:00
  • sar
  • TASK_RUNNING 実行可能待機(CPU負荷) => ロードアベレージに計上される
  • TASK_INTERRUPTIBLE 割込可能待機(入力待ち等)
  • TASK_UNINTERRUPTIBLE 割込不能待機(IO負荷) => ロードアベレージに計上される
  • TASK_STOPPED 実行中断
  • TASK_ZOMBIE 実行不能

# sar -qのカラム

  • runq => 実行可能待機プロセス数
  • plist => プロセス+スレッド総数
  • ldavg1 => ロードアベレージ(1分間)
  • ldavg5 => ロードアベレージ(5分間)
  • ldavg15 => ロードアベレージ(15分間)

# sar -wのカラム

proc => 秒間生成プロセス数 cswch => 秒間コンテキストスイッチ数
